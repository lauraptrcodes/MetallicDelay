# MetallicDelay
<img width="790" height="405" alt="Screenshot 2026-03-04 114756" src="https://github.com/user-attachments/assets/bbd9ba12-9e91-4675-a823-e150d63b117c" />

MetallicDelay implements a nonlinear recursive delay structure inspired by physical modeling waveguides and saturated feedback systems.
At high feedback values, the nonlinear delay structure exhibits complex dynamical behaviour.
Due to the recursive nonlinearity, the system may enter regimes similar to discrete-time chaotic systems, where small variations in input or internal state result in large spectral differences.
This behaviour is characteristic of nonlinear recursive systems described in physical modeling literature.
